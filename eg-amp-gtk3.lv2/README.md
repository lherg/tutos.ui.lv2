# eg-amp-gtk3.lv2<br/>
Makefile only works for linux.<br/>
A copie of eg-amp from  lv2 distribution.<br/>
Adding a gtk3 UI.<br/>
![alt text](https://github.com/lherg/tutos.ui.lv2/blob/main/png/eg-amp-gtk3.png)<br/>
<br/>

Compile<br/>
```
make
```

Install<br/>
```
make install
```

Run<br/>
```
jalv.gtk http://lv2plug.in/plugins/eg-amp-gtk3
```
